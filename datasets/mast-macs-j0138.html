Dataset: MACS J0138 — MAST/HST (First Pass)

Exact files and provenance used to build the lensing cutouts and NFW sweep for MACS J0138.0−2155.
Last updated: 2025‑09‑03

Overview

This page inventories the precise inputs used for the first-pass lensing setup: HST cutout FITS for IR/optical filters, preview PNGs, BCG center, and the outputs from the NFW sweep and BCG profile fits. It is intended to be reproducible and auditable. Links below assume these files are published under /data/macs_j0138 on the site; local provenance under ~/ratio_test/data is also shown.

How to reproduce

  1. Start from the local HST DRZ/DRC FITS in ~/ratio_test/data/mast_raw/HST/* and generate cutouts centered on the BCG (see bcg_center.txt).
  2. Record pixel scales/WCS from headers; verify with preview PNGs.
  3. Run the NFW grid sweep to solve θE (or accept external M(<R*)) and output:
    • nfw_rs_rhos_grid.csv, sweep_summary.csv
    • nfw_profile.csv (representative/best‑fit)
    • nfw_run_summary.txt (captures zL, zS, scale, θE)
  4. Fit the BCG radial profile and store radial_profile_kpc.csv and sersic_fit.txt.
  5. Publish the cutouts and outputs under /data/macs_j0138; keep raw MAST trees under /data/mast_raw (optional, large).

Notes

  • Preview PNGs (same stems as FITS) are included for quick visual QA but are not used in computations.
  • Local angles/distances reported here are for this run only; always check nfw_run_summary.txt if you re‑run the sweep.
  • JWST NIRSpec files listed in provenance were not used in this MACS J0138 first pass.
